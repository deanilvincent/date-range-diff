# Overview
Basic simple NPM package in getting date range difference between 2 specific dates.

# Installation
`npm i date-range-diff --save`

# Basic Setup & Usage

```
const dateDiff = require("date-range-diff");

console.log(dateDiff("2020-01-26", "2020-01-24").days)
// returns 2
```
